스크립트에서 color값을 활용하여 색상을 적용시키기 위해 사용하는 것.

스크립트로 선택한 값을 가져가기 위해서는 아래와 같이 여러가지 방법이 있다.

1. 태그 이름을 이용 :
1) document.getElementsByTagName('input') => 모든 input을 다 가져와라
2) document.querySelector('input') => 맨 위에꺼만 가져와라
3) document.querySelectorAll('input') => input다 가져와라

2. name 속성을 이용 :
1)document.form이름.input에 있는 name속성 ex)document.ff.color
2)document.getElementsByName('color')

3. id 속성을 이용 :
1) document.getElemnetById('color')
2) document.selector('#color')

4. class 속성을 이용 :
1) document.getElementByClassName('color')
2) document.querySelector('.color')
3) document.querySelectorAll('.color')
