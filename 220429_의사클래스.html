상태에 따른 스타일을 지정한다.
 a:link =>처음 만들어졌을 때 /a태그에서만 가능하다. 
 a:visited =>클릭해서 해당 사이트에 방문하고 나서  / a태그에서만 가능하다.
 a:hover => 마우스를 올려놨을 때  /a뿐만아니라 p,br,span등 다른 태그에서도 사용 가능하다.
 a:active => 마우스를 누르고있는 상태일 때 /a뿐만아니라 p,br,span등 다른 태그에서도 사용 가능하다.
