	<button type = "button">확인</button>
	==>
 	type="button"인 요소는 특별한 기능이 없다.
	그렇기 때문에 script를 이용하여 기능을 부여해주어야 한다.
	
	onclick = "함수이름()" : 클릭 이벤트로 정의된 함수를 호출한다.
	==>
	[함수정의]
	=>return은 있어도 되고 없어도 되고..
	=>무조건 function으로 씀.
	1) 파라미터가 없을 때 : function aa(){     return 100;}
	2) 파라미터가 있을 때 : function aa(x){                 }
	<input type = "button" value = "확인" onclick ="">
	<button type = "button" onclick = "">확인</button>
